Introduction

Python is a high-level, general-purpose programming language. Its design philosophy emphasizes code readability with the use of significant indentation.

Python is dynamically typed and garbage-collected. It supports multiple programming paradigms, including structured (particularly procedural), object-oriented and functional programming. It is often described as a "batteries included" language due to its comprehensive standard library.Python can be used for many different types of programming needs such as:

  • Data analysis
  • Game development
  • Machine learning

Pythons Creation

Python was invented in the late 1980s by Guido van Rossum at Centrum Wiskunde & Informatica (CWI) in the Netherlands as a successor to the ABC programming language, which was inspired by SETL, capable of exception handling and interfacing with the Amoeba operating system. Its implementation began in December 1989. Van Rossum shouldered sole responsibility for the project, as the lead developer, until 12 July 2018, when he announced his "permanent vacation" from his responsibilities as Python's "benevolent dictator for life" (BDFL), a title the Python community bestowed upon him to reflect his long-term commitment as the project's chief decision-maker (he's since come out of retirement and is self-titled "BDFL-emeritus"). In January 2019, active Python core developers elected a five-member Steering Council to lead the project.

Python can be run on Linux, macOS, and Windows operating systems. Numerous versions of it have been released periodaclly since its creation in the 1980s.

Python currently is one of the most popular programming languages due to its versatility. It is also fairly quick to learn with many websites offering free practice labs.

If Statements

The if statement, which conditionally executes a block of code, along with else and elif (a contraction of else-if).

If statements can be used in conditional instances, such as outputting a specific response if a certain variable is true.

For Statements

The for statement, which iterates over an iterable object, capturing each element to a local variable for use by the attached block. It acts as an iterator method. Let's see what the for statment looks like

fruits = ["apple", "banana", "cherry"] for x in fruits: print(x)

For loops do not require indexing variables.

While Statements

While loops can be executed when the statement is true.

i = 1 while i < 6: print(i) i += 1

When using while loops, the relevant variables must be ready.

Try statements

The try block lets you test a block of code for errors.

try: print(x) except: print("An exception occurred")

The try block is used in tandem with the except block to output text or different variables in the case of exceptions.
